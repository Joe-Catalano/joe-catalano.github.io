IQ-Leaf System

This project involved creating an end-to-end IoT solution for plant care...

Key Features:

  • Real-time capacitive moisture sensing.
  • Automated watering via 5V pump and relay control.
  • Custom PCB designed in Eagle, integrating ESP32-S3, sensors, and drivers.
  • Web-based dashboard (HTML/CSS/JS) served from ESP32 for remote monitoring and control.
  • ESP32-CAM integration for live visual feedback and time-lapses (potential feature).
  • Local TFT LCD display showing current status, sensor readings, and IP address.
  • Code primarily in C (Arduino framework), with HTML/CSS/JS for the web interface.

Technologies Used: ESP32-S3, C/C++, Arduino Framework, HTML, CSS, JavaScript, Eagle (PCB), Capacitive Moisture Sensor, ESP32-CAM, TFT LCD.
