IQ-Leaf System

This project involved creating an end-to-end IoT solution for plant care, integrating sensors, actuators, custom hardware, and both local and remote interfaces. It showcases skills in embedded systems programming, PCB design, web development (for the interface), and IoT principles.

Key Features & Implementation Details:

  • **Real-time Sensing:** Utilized a capacitive moisture sensor connected to the ESP32's ADC for accurate soil hydration monitoring, less prone to corrosion than resistive sensors.
  • **Automated Actuation:** Controlled a 5V water pump via a relay module, triggered by software logic based on configurable moisture thresholds. Implemented safeguards to prevent overwatering.
  • **Custom Hardware:** Designed and fabricated a custom PCB using Eagle CAD software. This integrated the ESP32-S3 microcontroller, sensor interfaces, power regulation (3.3V for ESP, 5V for peripherals), relay driver, and connectors for a modular design.
  • **Remote Interface:** Developed a responsive web dashboard served directly from the ESP32 using an asynchronous web server library. This allowed monitoring sensor data (moisture, potentially temperature/light later) and manual pump control over the local Wi-Fi network via HTML, CSS, and JavaScript.
  • **Visual Monitoring:** Integrated an ESP32-CAM module streaming video over Wi-Fi, accessible through the web interface, allowing for visual checks on the plant's condition.
  • **Local Display:** Included a TFT LCD screen connected via SPI to display current moisture levels, system status (e.g., "Watering", "Idle"), Wi-Fi connection details, and IP address for easy access to the web interface.
  • **Programming:** Core logic implemented in C/C++ within the Arduino framework for the ESP32. Leveraged libraries for Wi-Fi connectivity, web server, sensor communication (I2C/Analog), display drivers (SPI), and camera interfacing.

Technologies Used: ESP32-S3, C/C++, Arduino Framework, HTML, CSS, JavaScript, Eagle CAD (PCB), Capacitive Moisture Sensor, ESP32-CAM, TFT LCD (SPI), Relay Module, Wi-Fi, Async Web Server, Git.
